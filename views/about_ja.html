About

Go Walker は、Bitbucket、Github、Launchpad、そして、Google Project Hosting 上にある Go プロジェクトのドキュメントを表示します。

Go Walker 自体のソースコードは、Github上にあります。

Go Walker へプロジェクトを追加するには

Go WalkerはGoのソースコードからドキュメントを生成します。godoc ツール用に記述するドキュメントの ガイドライン が、プロジェクトのパッケージ間の関係を表すためにGo Walkerへ適用されます。

パッケージの最初に、パッケージの要約を記述しておくことが重要です。Go Walkerは、プロジェクトの一覧でその最初の文章を表示します。

Go Walkerへプロジェクトを追加するには、importパス名でプロジェクトを search から検索します。もしGo Walkerにプロジェクトのドキュメントが無ければ、Go Walkerは自動的にリポジトリへソースコードを取りに行き、ドキュメントを追加します。追加には5秒程度要します。

Go Walkerはパッケージのアップデートを毎日確認します。すぐにドキュメントを更新したいなら、パッケージドキュメントの最下部にあるRefreshリンクをクリックしてください。

Go Walker からパッケージを削除するには

Go Walkerは、パッケージのアップデートを確認する際、リポジトリ上からパッケージが削除されていると自動的にドキュメントを削除します。すぐにドキュメントを削除したいなら、パッケージドキュメントの最下部にあるRefleshリンクをクリックしてください。だたし、他のパッケージとの依存がある場合は削除されないことに注意してください。

もし、Go Walkerへあなたのパッケージのドキュメントを表示して欲しくないような場合は、削除したいパッケージの import パスと一緒にIssueを登録してください。

Add or remove tags for your project

Like we show the different categories in Index page, you can also label your project by appending tags to URL or press g + t to call control panel in documentation page. For example, Qbs is a ORM, so to lable it by typing github.com/coocood/qbs:tag=orm in the search box, or just visit gowalker.org/github.com/coocood/qbs:tag=orm, and use :rtag= to remove tags from your project.

翻訳のお手伝い

Go Walker はいくつかの言語での表示をサポートしています。github にある"view"ディレクトリにある html を翻訳してpull request して頂けると幸いです。
