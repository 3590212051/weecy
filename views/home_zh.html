什么是 Go 步行者？

Go 步行者是一个用于在线生成并浏览 Go 项目 源码 文档的 Web 服务器，目前仅支持 Bitbucket、Github、Google Project Hosting 和 Launchpad 四大版本控制系统。该服务器所产生的 不只是 API 文档， 如果您仅希望查看此类文档，GoDoc 才是最合适的地方。

如果您想了解如何向 Go 步行者添加开源项目或查看所有给力的功能，请仔细阅读 关于 页面。
