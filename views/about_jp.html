About

Go Walker は、Bitbucket、Github、Launchpad、そして、Google Project Hosting上にあるGoプロジェクトのドキュメントを表示します。

Go Walker自体のソースコードは、Github上にあります。

Go Walkerへプロジェクトを追加するには

Go WalkerはGoのソースコードからドキュメントを生成します。godocツール用に記述するドキュメントのガイドラインが、プロジェクトのパッケージ間の関係を表すためにGo Walkerへ適用されます。

パッケージの最初に、パッケージの要約を記述しておくことが重要です。Go Walkerは、プロジェクトの一覧でその最初の文章を表示します。

Go Walkerへプロジェクトを追加するには、importパス名でプロジェクトをsearchから検索します。もしGo Walkerにプロジェクトのドキュメントが無ければ、Go Walkerは自動的にリポジトリへソースコードを取りに行き、ドキュメントを追加します。追加には5秒程度要します。

Go Walkerはパッケージのアップデートを毎日確認します。すぐにドキュメントを更新したいなら、パッケージドキュメントの最下部にあるRefreshリンクをクリックしてください。

Go Walkerからパッケージを削除するには

Go Walkerは、パッケージのアップデートを確認する際、リポジトリ上からパッケージが削除されていると自動的にドキュメントを削除します。すぐにドキュメントを削除したいなら、パッケージドキュメントの最下部にあるRefleshリンクをクリックしてください。だたし、他のパッケージとの依存がある場合は削除されないことに注意してください。

もし、Go Walkerへあなたのパッケージのドキュメントを表示して欲しくないような場合は、削除したいパッケージのimportパスと一緒にIssueを登録してください。

タグの追加

Indexページでは異なるカテゴリでも表示します。URLの最後にタグを追加することでタグ付けすることができます。たとえば、Qbsパッケージは"ORM"(タグについては後述)ですので、github.com/coocood/qbs:tag=ormをSearchボックスへ入れるか、直接gowalker.org/github.com/coocood/qbs:tag=ormへアクセスします。また、:rtag=を使うことでタグをプロジェクトから取り消すことができます。現在、"wf"(Web Framework)、"orm"()Object Relation Mapping)、"dbd" (Database Driver)、"gui"(Graphic User Interface)、"net"(Networking)をサポートしています。コロン:でタグを区切ることで次のように複数利用できます：:tag=orm:wf:dbd

翻訳のお手伝い

Go Walkerはいくつかの言語での表示をサポートしています。githubにある"view"ディレクトリにあるhtmlを翻訳してpull requestして頂けると幸いです。
