关于

Go 步行者用于为托管在版本控制系统上的 Go 开源项目生成在线 API 文档，目前支持的平台包括 Bitbucket，Github，Google Code，Launchpad 和 Git @ OSC。

您可以在 Github（稳定版）或 Git @ OSC（测试版）上浏览并下载该项目的源代码。

添加新的项目到 Go 步行者

Go 步行者是一个用于生成 Go 项目 API 文档的服务器，有关文档注释规范，请参考官方给出的 大纲。Go 步行者兼容所有 godoc 工具所支持的标准，同时 Go 步行者还将会显示类型在不同包之间的关联。

对于一个具有良好可读性的项目而言，规范完整的文档说明是极其重要的。Go 步行者会截取每个项目的包的首行文字作为该包的纲要，并在搜索项目时展现给阅读者。

如果您打算将某个项目添加到 Go 步行者，可以通过 搜索 该项目的导入路径来通知服务器。如果该项目的文档已经存在，则将直接呈现给您，否则会从相关的版本控制系统上即时下载并生成文档。如果您的操作是添加新的项目，则最多可能需要 5 秒的时间来完成，因为 Go 步行者会将源码和文档展示在相同页面上。

Go 步行者每天都会检查各个项目的更新情况，你也可以通过底部的刷新按钮来进行强制刷新。

从 Go 步行者中删除项目

Go 步行者将会在每日检查更新时自动删除更新失败的项目，您也可以在删除托管在版本控制系统上的项目后，通过强制刷新的方式来完成删除操作。

如果您只是单纯的想要禁止他人通过本服务器产生您的项目源码文档，请通过 Oepn Issue 的方式提出。

控制面板

Go 步行者提供控制面板以方便页面上的操作，在文档页面按下问号 "?" 即可打开控制面板。感谢 chenwenli 提供的帮助！

为您的项目增加或移除标签

正如我们在 标签 页面向您展示不同分类的项目那样，您可以通过为您的项目添加标签来进行归类展示，只需要在您项目导入路径后加上标签搜索或在文档页面按下 g + t。例如，Qbs 是一个 ORM，因此可以通过搜索以下内容来对它进行标记：github.com/coocood/qbs:l=orm，您也可以通过访问页面 gowalker.org/github.com/coocood/qbs:l=orm 来完成标记操作。如果您想要删除某个标记，则使用 :rl=。

协助站点翻译

Go 步行者支持多语言的网站用户界面。您可以通过翻译 "views" 目录下的文件来帮助站点支持您所使用的语言。

多语言文档

如果您的项目目录包含 readme(.md) 文件，Go 步行者将会使用 readme(.md) 作为英文版本，readme_[zh|cn](.md) 作为中文版本，并根据用户语言选项自动展示它们。

书签工具

Go 步行者书签工具通过 JavaScript 脚本来动态获取您当前在代码托管平台上浏览的项目地址，并自动跳转到本网站的相关页面。请将该链接保存至您的收藏夹来使用此功能：Go 步行者 - 书签.
