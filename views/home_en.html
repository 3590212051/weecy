What is this?

Go Walker is a web server that generates Go projects API documentation with source code on the fly from projects on Bitbucket, Github, Google Code, Launchpad, Git @ OSC and CSDN Code.

Main features including Code View, Link Jump, etc.

Please read the About Page for information about adding projects to Go Walker and more.
