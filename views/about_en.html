About

Go Walker displays documentation for Go projects on Bitbucket, Github, Launchpad and Google Project Hosting.

The source code for Go Walker is available on Github.

Add a project to Go Walker

Go Walker generates documentation from Go source code. The guidelines for writing documentation for the godoc tool apply to Go Walker and display object relationships between packages in your project.

It's important to write a good summary of the package in the first sentence of the package comment. Go Walker indexes the first sentence and displays the sentence in project lists.

To add a project to Go Walker, search for the peoject by import path. If Go Walker does not already have the documentation for the project, then Go Walker will fetch the source from the version control system on the fly and add the documentation. It may take up to 5 seconds because source code are contained in same page also.

Go Walker checks for package updates once per day. You can force Go Walker to refresh the documentation immediately by clicking the refresh link at the bottom of the package documentation page.

Multi-language Documentation

Go Walker supports more than one language for your source code documentation, and you don't need to change anything you've done so far, it automatically changes documentation language when user change the language option in the bottom of the page. If you want to use this feature, see Multi-language Documentation Support.

Remove a package from Go Walker

Go Walker automatically removes packages deleted from the version control system when Go Walker checks for updates to the package. You can force Go Walker to remove a deleted package immediately by clicking the refresh link at the bottom of the package documentation page, but be aware that this operation will not delete dependencies packages in case other packages use some of them.

If you do not want Go Walker to display documentation for your package, please open an issue with the import path of the path of the package that you want to remove.

Add tags for your project

Like we show the different categories in Index page, you can also label your project by appending tags to URL and search. For example, Qbs is a ORM, so to lable it by typing github.com/coocood/qbs:tag=orm in the search box, or just visit gowalker.org/github.com/coocood/qbs:tag=orm, and use :rtag= to remove tags from your project. For now, we support "wf" for Web Framework, "orm" for Object Relation Mapping, "dbd" for Database Driver, "gui" for Graphic User Interface, "net" for Networking, you can use : to break up multiple tags, like :tag=orm:wf:dbd

Help translate site to your language

For now, Go Walker supports two languages for site user interface, which are Chinese and English. You can translate files under folder "views" to help Go Walker support your language.
